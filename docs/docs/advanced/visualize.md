# Dataset Visualization

Deep Lake has a web interface for visualizing, versioning, and querying machine learning datasets. It utilizes the Deep Lake format under-the-hood, and it can be connected to datasets stored in all Deep Lake [storage locations](../../getting-started/storage-and-creds/).

![Visualizer](images/visualize.jpg)

## Visualization

To visualize your dataset, you need to create an account in the [Deep Lake web UI](https://app.activeloop.ai/). Once you have an account, you can create a new dataset and upload your data. Alternatively you can connect the dataset in your storage location to the Deep Lake web UI.

The datasets created in the Deep Lake web UI have activeloop url in the format `al://organization/dataset`. To visualize your dataset please find the dataset in your organization profile or directly visit the `https://app.activeloop.ai/organization/dataset`.
